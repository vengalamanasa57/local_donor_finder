# LifeLink: Real-Time Blood Assistance Network 

LifeLink is a specialized emergency response application designed to bridge the gap between blood seekers and donors, providing a critical link during medical emergencies.

##  Key Features
* **One-Tap SOS**: Instantly broadcast a blood request with real-time data synchronization.
* **Live Emergency Map**: Interactive interface to visualize and track active blood requests in the vicinity.
* **Direct Connectivity**: Integrated "I Can Help" feature to connect donors to seekers via instant calling.
* **Regional Verification**: Optimized for localized response to ensure high-speed donor prioritization.

##  Tech Stack
* **Framework**: Flutter
* **Database**: Firebase Firestore
* **Maps**: Google Maps SDK
* **Cloud Services**: Google Cloud Platform

##  Note on Geolocation
The current MVP is configured for the Warangal regional corridor to demonstrate stable real-time data flow. While the core engine supports global GPS, we have implemented a regional label for the demo to showcase verified local emergency response.
